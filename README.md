# Serve-U: Services Providing Application

Serve-U is a digital platform designed to connect customers with verified local service providers for home and vehicle-related services.

## Tech Stack
- **Web & Backend:** [Next.js](https://nextjs.org/) (TypeScript, Tailwind CSS)
- **Mobile:** [React Native Expo](https://expo.dev/)
- **Database:** [PostgreSQL](https://www.postgresql.org/) with [Prisma ORM](https://www.prisma.io/)
- **AI Integration:** Google Vision API for Provider Identity Verification
- **Payments:** Secure Online Payment Integration (e.g., PayFast)
- **Storage:** Cloud Storage (AWS S3 / Google Cloud Storage)

## Core Features
- User and Service Provider Registration & Authentication
- AI-based Provider Identity Verification (OCR)
- Service Listing and Categorization (Home, Vehicle)
- Booking Management (Create, Accept/Reject, Track Status)
- Secure Online Payments
- Push Notifications
- AI-based Service Recommendations
